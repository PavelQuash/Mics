# Описание

## Железо

- Микроконтроллер Arduino Nano v3 ATmega328
- Модуль часов реального времени DS1302
- Датчик температуры и влажности DHT22
- Реле 
- Детектор газа MQ135
- Датчик давления GY-68 BMP180
- Датчик света 

## Features

0. Измерение температуры, влажности, CO2, давления, освещенности    
0. Фиксации времени измерения  
0. Включения вентилятора и увлажнителя воздуха для регулировки влажности  
0. Включение обогревателя для регулировки температуры воздуха
